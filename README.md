# InfoHub-Challenge

A full-stack application with React frontend and Node.js/Express backend.

***

## Features

- Weather information module
- Currency converter module
- Random quote generator

***

## Project Structure

- `client/` : React frontend
- `api/` : Node.js/Express backend

***

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/infohub-challenge.git
   ```

2. Install backend dependencies:
   ```bash
   cd api
   npm install
   ```

3. Install frontend dependencies:
   ```bash
   cd ../client
   npm install
   ```

4. Set up environment variables:
   - Add your `.env` file in the `server` folder with necessary keys.

***

## Usage

- Start the backend server:
  ```bash
  cd api
  npm start
  ```
- Start the frontend app:
  ```bash
  cd ../client
  npm start
  ```
